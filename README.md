a dps calculator for the secret world.
based  on data from https://docs.google.com/spreadsheets/d/1ZrGxfQVekLHIYyn0sysZSYQOdDzKEHRGLDE36BcJ6q0/edit?gid=0#gid=0
shoutout to ttg and whoever else made those documents.
active and passive scaling has been taken from rdb data.

known issues:

penetration% has to be entered manually for now, i dont know the penetration rating to % scaling yet.
defensive stats dont work yet, i need more data.
